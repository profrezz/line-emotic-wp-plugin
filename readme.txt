Lind-emotic Wordpress plugin 
https://github.com/profrezz/line-emotic-wp-plugin
This is free plugin for Wordpress with ability to download official Line emoticon to express your emotiion of each WP's articles.
 
How to use
1. create folder name "line-emotic" in "wp-content\plugins" directory 
2. copy whole files into that
3. login Wordpress and enable plugin 
4. go to setting page & download new emoticon
5. ready to use

Hope you enjoy!
